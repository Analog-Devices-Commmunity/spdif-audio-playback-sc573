1. Description

This example can play back audio from a receive SPDIF PCM audio input via the ADAU1962a DAC on the ADSP-SC573 EZ-Kit.

2. Processor

  • ADSP-SC573

3. Overview

The SPDIF playback uses the receive SPDIF interface as a source of a stereo I2S PCM audio stream. The input audio typically has a sampling rate of 48kHz and the AD1962a is configured to play back the audio at 96kHz. In spdif_playback.h there is a #define CALLBACK_COUNT that determines the playback duration of the example. By default it is set to run for more than 10 seconds.

5. External connections

Connect a SPDIF audio input to the input connector of SPDIF in/out J12 (red connector).

The two channel playback audio can be heard on the audio output connectors of J11 shown as OP1 and OP2 in the block diagram provided.
